1 Python and Jupyter Notebook

Python is a modern general purpose programming language with many external packages (such as NumPy, SciPy, Pandas, and Scikit-learn) for data analytics.

Jupyter notebook connects to the Python engine to provide an interactive session (similar to the R Console) within a web browser for writing and executing codes as well as displaying plots.

1.1 Installation

We will use the Anaconda Python instead of using the base Python from the official website https://www.python.org/. The advantage of the Anaconda distribution of Python is that it comes with the package manager conda for easier installation of external packages and handling of their dependencies. Moreover, the full Anaconda distribution comes pre-installed with a list of the most useful packages for data analytics.

You have two choices. You can either install the full Anaconda distribution (600MB+), or install the minimal Miniconda (about 50MB) and then install the needed external packages later.

Anaconda

  1. Go to the Anaconda website https://www.anaconda.com/ â Products â Individual Edition â Download
  2. For Windows, download Python 3.8 version either the 64-bit or 32-bit Graphical Installer depending on your CPU. (Most modern computers would probably be 64-bit.) Double-click the downloaded .exe file to install.
  3. For MacOS, download Python 3.3 version either the Graphical Installer or Comnand-Line Installer. Follow the instruction on that page to continue with the downloaded file.

Miniconda

  1. Go to the Miniconda website https://docs.conda.io/en/latest/miniconda.html
  2. For Windows, select either the Python 3.8 64-bit or 32-bit exe installer depending on your CPU. (Most modern computers would probably be 64-bit.) Double-click the downloaded .exe file to install.
  3. For MacOSX, select the Python 3.8 64-bit bash installer. Double-click the downloaded .sh file to install.
  4. Go to the Anaconda Prompt (in Windows, might need to Run as administrator) or Terminal (in Mac OS X).
  5. In the command prompt, type conda update --all to ask the package manager conda to update all the existing packages.
  6. Type conda install jupyter matplotlib numpy pandas scikit-learn scipy to install the essential external packages for data analytics.

Note that we are using Python 3.6+ but not the older Python 2.7. Here are some differences between Python 2 vs Python 3:

  • Package dependency
  • Integer division e.g.Â 1 / 2 equals 0 in Python 2 but 0.5 in Python 3
  • Print: print "hello" vs print("hello")
  • Iterator: range() vs xrange()

1.2 Verifying Python and Conda

If Anaconda or Miniconda is installed successfully, you can run the conda package manager inside the Anaconda Prompt (in Windows) or Terminal (in Mac OS X). To start the command prompt:

To start a Jupyter notebook session, go to the command prompt:

  1. Type jupyter notebook at the command prompt
  2. It should open a new webpage in your web browser with the URL http://localhost:8888/tree and the page should look like a file browser.
  3. To shut down the Jupyter notebook session, click the command prompt window to bring it into focus, and press Ctrl+C (in Windows) or control+C (in Mac OS X).
  4. The webpages in the browser can now be closed.

If the steps above are completed without errors, then you have successfully run a Jupyter notebook session and is ready to use Python interactively!

4.6 Time Complexity and Big O Notation

  • Time complexity is the analysis of the runtime of an algorithm and how it scales with the size of the input data.
  • The big O notation is a standard way to represent the time complexity.
  • For example, the amount of time to retrieve an element from a sequence using integer indexing does not depend on the length \(n\) of the sequence. Therefore, it has constant time complexity \(O(1)\).
x = [1, ..., n]
x[n] # O(1)
  • For example, computing the sum of a sequence x of length \(n\) requires \(O(n)\) number of operations i.e.Â it scales linearly with the input size.
x = [1, ..., n]
total = 0
for v in x:
    total += v
  • For example, computing the sum of the entries of an \(n\times n\) matrix m requires \(O(n^2)\) number of operations.
total = 0
for i in range(n):
    for j in range(n):
        total += m[i,j]
  • For example, the matrix multiplication of two \(n\times n\) matrices requires \(O(n^3)\) number of operations (because the product has \(n^2\) entries and each requires a dot-product which is \(O(n)\) number of operations).
  • A lot of the matrix decomposition algorithms (e.g.Â LU, QR, Cholesky, and SVD) are \(O(n^3)\).
  • It is a fact that any (comparison-based) sorting algorithm requires at least \(O(n\log_2 n)\) number of comparisons (on average) to sort a sequence. However, the simple bubble sort requires \(O(n^2)\) comparisons and is therefore not optimal.
  • In general, an algorithm with time complexity \(O(n \log n)\) is reasonable, \(O(n^2)\) is not fast, and \(O(n^3)\) is slow and difficult to scale.

6.2 Escape Character

  • If the dot . has a special meaning in a regular expression pattern, then how can we describe a pattern to match the literal dot . itself? Answer: Combine with the escape character \.
  • An escape character combines the backslash character \ with a (typically alphabetical) character to represent a meaning that is different from the character itself.
  • To let Python interpret the plus sign . literally, use \. in the regular expression pattern.
import re

s = 'Please email analytics@uchicago.edu or grahamschool@uchicago.edu.'
re.findall('\w+@\w+\.\w+', s)
['analytics@uchicago.edu', 'grahamschool@uchicago.edu']
  • Regular expression pattern can be complicated. Here is an example involving more advanced syntax such as lookbehind assertion (?<=...) and grouping (?:...) to extract price tag i.e.Â numbers following the dollar sign with optional cents.
import re

s = 'Bought 2 oranges for $1.99 and 3oz of cheese for $5.'

re.findall('\d+', s) # Only match consecutive digits
['2', '1', '99', '3', '5']
re.findall('\d+\.\d{2}', s) # Only match number with 2 decimal places
['1.99']
re.findall('\d+\.?\d*', s) # Fail to exclude numbers without dollar sign and non-cent dot
['2', '1.99', '3', '5.']
re.findall('(?<=\$)\d(?:\.\d{1,2})?', s)
['1.99', '5']
